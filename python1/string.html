Tech Atival - String formátozás

Feladatok

1. Hozz létre egy "szo" nevű string változót, amely a "Python" szöveget tartalmazza! Írd ki a string hosszát a képernyőre!

2. Hozz létre két string változót, "keresztnev" és "vezeteknev" névvel, amelyek a saját nevedet tartalmazzák! Fűzd össze a két stringet, és írd ki az eredményt a képernyőre!

3. Hozz létre egy "karakter" nevű változót, amely a 'a' karaktert tartalmazza! Írd ki a karakter ASCII kódját a képernyőre!

4. Hozz létre egy "mondat" nevű string változót, amely tartalmaz egy rövid mondatot! Számítsd ki, hogy hányszor fordul elő benne a "a" betű, és írd ki az eredményt!

5. Hozz létre egy "jelszo" nevű string változót, amely tartalmaz egy tetszőleges jelszót! Ellenőrizd, hogy a jelszó tartalmazza-e a "!" karaktert, és írd ki az eredményt (igaz/hamis)!

6. Hozz létre egy "szoveg" nevű string változót, amely tartalmaz egy rövid szöveget! Írd ki a szöveg első karakterét és az utolsó karakterét a képernyőre!

7. Hozz létre egy "szoveg" nevű string változót, amely egy mondatot tartalmaz! Konvertáld az összes karaktert nagybetűssé, majd írd ki az eredményt a képernyőre!

8. Hozz létre egy "szoveg" nevű string változót, amely egy mondatot tartalmaz! Vágd le az első 5 karaktert, és írd ki a maradék szöveget a képernyőre!

9. Hozz létre két string változót, "elso" és "masodik" névvel, amelyek két szót tartalmaznak! Hasonlítsd össze a két stringet, és írd ki, hogy egyenlőek-e (igaz/hamis)!

10. Hozz létre egy "szoveg" nevű string változót, amely tartalmaz egy mondatot! Ellenőrizd, hogy a mondat tartalmazza-e a "Python" szót, és írd ki az eredményt (igaz/hamis)!
