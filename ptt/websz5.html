Ismerkedés a JavaScripttel

A témakör elsajátítását követően a tanuló:

  1. Ismeri a JavaScript nyelv szerepét, fontosabb tulajdonságait.
  2. Képes JavaScript kód beágyazására weboldalba.
  3. Képes „Hello World!” típusú alkalmazás készítésére alert() függvény segítségével.
  4. Képes külső fájlban elhelyezett JavaScript kód csatolására a weboldalhoz.

Csak az önálló tanulás bír tartós értékkel. • John Taylor Gatto
