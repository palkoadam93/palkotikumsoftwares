Hibakeresés weboldalakon, verziókezelő és csoportmunka-eszközök

A témakör célja, hogy a tanuló képessé váljon a HTML-oldalak hibakeresési eszközei, a fejlesztést támogató csoportmunka-eszközök, valamint a Git verziókezelő rendszer használatára. A témakör elsajátítását követően a tanuló:

  1. Ismer legalább egy, a HTML-oldalak validációjára szolgáló eszközt (pl. HTML Validator for Chrome, W3C Markup Validation Service, Nu HTML5 Validator) és képes annak használatára.
  2. Ismeri a böngészőprogramok beépített fejlesztőeszközeinek alapvető szolgáltatásait és képes azok gyakorlati használatára (javasolt eszköz: Chrome DevTools).
  3. Képes a Git verziókezelő rendszer helyi telepítésére, ismeri a Git célját és alapvető szolgálatásait.
  4. Képes a Git verziókezelő rendszer helyi telepítésére, ismeri a Git célját és alapvető szolgálatásait.
  5. Képes használni a lokális repository kezelésére szolgáló legfontosabb parancsokat (init, add, commit, branch, checkout, merge, status).
  6. Képes használni a lokális repository kezelésére szolgáló legfontosabb parancsokat (init, add, commit, branch, checkout, merge, status).
  7. Képes a GitHub szolgáltatás használatával távoli (remote) repositoryk létrehozására és kezelésére,
  8. tudja használni a távoli repositoryk kezelésével összefüggő legfontosabb Git-parancsokat (clone, pull, push).
  9. Ismeri a csoportmunkában végzett fejlesztést támogató online eszköz legfontosabb szolgáltatásait és képes annak használatára (javasolt eszköz: Slack).
  10. Ismeri a csoportmunkában végzett fejlesztést támogató online eszköz legfontosabb szolgáltatásait és képes annak használatára (javasolt eszköz: Slack).

Csak az önálló tanulás bír tartós értékkel. John Taylor Gatto
