Tuple

SuliPY - Tuple adattípus

Tech Atival - Tuple

Feladatok

1. Írj egy programot, amely létrehoz egy tuple-t 5 tetszőleges egész számmal. A program írd ki a tuple összes elemét egy sorban!

2. Írj egy programot, amely létrehoz egy tuple-t, amely tartalmaz 4 tetszőleges szót. A program írd ki a tuple harmadik elemét!

3. Írj egy programot, amely létrehoz egy tuple-t 6 véletlenszerű lebegőpontos számmal. A program számolja ki és írd ki a tuple összes elemének átlagát!

4. Írj egy programot, amely létrehoz egy tuple-t, amely tartalmaz 5 tetszőleges számot és szót. A program készítsen egy új tuple-t, amely csak a szavakat tartalmazza az eredeti tuple-ból!

5. Írj egy programot, amely létrehoz egy tuple-t, amely tartalmaz 3 tetszőleges szót. A program adjon hozzá egy új szót az eredeti tuple végéhez, és írd ki a frissített tuple-t!

6. Írj egy programot, amely létrehoz egy tuple-t 8 tetszőleges egész számmal. A program határozza meg és írd ki a tuple legnagyobb és legkisebb elemét!

7. Írj egy programot, amely létrehoz egy tuple-t, amely tartalmaz 5 tetszőleges szót. A program írd ki a tuple minden elemét, fordított sorrendben!

8. Írj egy programot, amely létrehoz egy tuple-t, amely tartalmaz 6 véletlenszerű egész számmal. A program számolja meg, hányszor fordul elő egy adott szám a tuple-ban!

9. Írj egy programot, amely létrehoz egy tuple-t, amely tartalmaz 4 szót, mindegyik szót nagybetűs formában. A program készítsen egy új tuple-t, amely tartalmazza az összes szót kisbetűs formában!

10. Írj egy programot, amely létrehoz egy tuple-t, amely tartalmaz 5 tetszőleges egész számot. A program határozza meg és írd ki a tuple középső elemét!
