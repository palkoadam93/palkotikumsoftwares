Halmaz

SuliPY - Halmazok (set)

SuliPy - Metszet- és unióképzés

Feladatok

1. Írj egy programot, amely létrehoz két halmazt, például halmaz1 = {1, 2, 3, 4} és halmaz2 = {3, 4, 5, 6}. A program határozza meg és írd ki a két halmaz metszetét!

2. Írj egy programot, amely létrehoz két halmazt, például halmaz1 = {'alma', 'banán', 'cseresznye'} és halmaz2 = {'banán', 'dinnye'}. A program határozza meg és írd ki a két halmaz unióját!

3. Írj egy programot, amely létrehoz egy halmazt, amely tartalmaz 5 tetszőleges számot. A program adjon hozzá egy új számot a halmazhoz, majd írd ki a frissített halmazt!

4. Írj egy programot, amely létrehoz egy halmazt, amely tartalmaz 6 szót. A program távolítson el egy adott szót a halmazból, majd írd ki a frissített halmazt!

5. Írj egy programot, amely létrehoz két halmazt, például halmaz1 = {1, 2, 3, 4} és halmaz2 = {2, 4, 6, 8}. A program határozza meg és írd ki a halmazok különbségét (azokat az elemeket, amelyek csak az első halmazban találhatóak)!

6. Írj egy programot, amely létrehoz két halmazt, például halmaz1 = {10, 20, 30} és halmaz2 = {15, 20, 25}. A program határozza meg és írd ki a halmazok szimmetrikus különbségét (azokat az elemeket, amelyek csak az egyik halmazban találhatók, de nem mindkettőben)!

7. Írj egy programot, amely létrehoz egy halmazt, amely tartalmaz 7 tetszőleges számot. A program ellenőrizze, hogy egy adott szám szerepel-e a halmazban, majd írd ki az eredményt!

8. Írj egy programot, amely létrehoz két halmazt, például halmaz1 = {1, 2, 3} és halmaz2 = {1, 2, 3, 4, 5}. A program ellenőrizze, hogy az első halmaz egy része-e a második halmaznak (szubhalmaz-e)!

9. Írj egy programot, amely létrehoz egy halmazt, amely tartalmaz 5 tetszőleges szót. A program készítsen egy új halmazt, amely tartalmazza a szó első betűit az eredeti halmazban!

10. Írj egy programot, amely létrehoz két halmazt, például halmaz1 = {'a', 'b', 'c'} és halmaz2 = {'b', 'c', 'd'}. A program határozza meg és írd ki a halmazok unióját, metszetét, különbségét és szimmetrikus különbségét!
