Feladatok

1. Írj egy Python programot, amely létrehoz egy új `txt` fájlt, és beleír három sor szöveget. A program ezután olvassa be a fájl tartalmát, és írja ki a képernyőre!

2. Készíts egy Python programot, amely egy meglévő `txt` fájlba hozzáfűz egy új sort. A program ellenőrizze, hogy a fájl létezik-e, és ha nem, hozzon létre egy új fájlt!

3. Írj egy Python programot, amely beolvassa egy `txt` fájl tartalmát, és számolja meg, hány sor és hány karakter található a fájlban. A program írja ki ezeket az információkat!

4. Készíts egy Python programot, amely egy `csv` fájlt olvas be, és az első sor (fejléc) alapján írja ki a táblázat oszlopainak neveit!

5. Írj egy Python programot, amely egy `csv` fájlba ír be adatokat. A program kérjen be adatokat a felhasználótól, és minden új adatot új sorba írjon a fájlba!

6. Készíts egy Python programot, amely beolvassa egy `csv` fájl tartalmát, és minden sor tartalmát egy szótárban tárolja, ahol a kulcsok az oszlopnevek, az értékek pedig az adott sor értékei!

7. Írj egy Python programot, amely egy `txt` fájl tartalmát karakterenként olvassa be, és minden karaktert egy listába helyez el. A program írja ki a listát!

8. Készíts egy Python programot, amely egy `csv` fájl adatait módosítja. A program kérje be a felhasználótól, hogy melyik sort és oszlopot kell módosítani, és írja át a megadott adatot a fájlban!

9. Írj egy Python programot, amely egy `txt` fájl minden sorában keres egy adott szót, és írja ki azokat a sorokat, amelyek tartalmazzák a keresett szót!

10. Készíts egy Python programot, amely egy `csv` fájl sorait rendezi az egyik oszlop értékei alapján. A program kérje be az oszlop nevét, és az alapján rendezze a sorokat a fájlban!
