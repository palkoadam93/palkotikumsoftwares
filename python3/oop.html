Feladatok

1. Írj egy Python programot, amely definiál egy Person osztályt, amelynek van egy name és egy age attribútuma. Készíts egy példányt az osztályból, állítsd be az attribútumokat, és írd ki azokat!

2. Készíts egy Python programot, amely definiál egy Rectangle osztályt, amelynek van width és height attribútuma. Implementálj egy area() metódust, amely kiszámolja a téglalap területét, és írd ki a területet!

3. Írj egy Python programot, amely definiál egy Animal szülőosztályt és két alosztályt: Dog és Cat. A szülőosztály tartalmazzon egy make_sound() metódust, amely az alosztályokban felülírható. Hozz létre példányokat az alosztályokból, és hívd meg a make_sound() metódust!

4. Készíts egy Python programot, amely definiál egy BankAccount osztályt, amelynek van egy balance attribútuma. Implementálj deposit() és withdraw() metódusokat az egyenleg növelésére és csökkentésére, és teszteld a funkcionalitást!

5. Írj egy Python programot, amely definiál egy Vehicle osztályt, és implementálj benne egy start_engine() metódust. Készíts két alosztályt, Car és Motorcycle, és mindegyikben implementálj egy honk() metódust!

6. Készíts egy Python programot, amely definiál egy Student osztályt, amelynek van egy grades listája. Implementálj egy average_grade() metódust, amely kiszámítja és visszaadja a diák átlagos jegyét!

7. Írj egy Python programot, amely definiál egy Shape osztályt, és két alosztályt: Circle és Square. A szülőosztály tartalmazzon egy draw() metódust, amely az alosztályokban felülírható. Implementálj egy area() metódust minden alosztályban, amely kiszámítja a megfelelő területet!

8. Készíts egy Python programot, amely definiál egy Book osztályt, amelynek van title és author attribútuma. Implementálj egy display_info() metódust, amely kiírja a könyv címét és szerzőjét!

9. Írj egy Python programot, amely definiál egy Employee osztályt, amelynek van egy salary attribútuma. Implementálj egy give_raise() metódust, amely növeli a fizetést egy adott százalékkal!

10. Készíts egy Python programot, amely definiál egy Library osztályt, amelynek van egy books listája. Implementálj egy add_book() metódust, amely hozzáad egy könyvet a listához, és egy list_books() metódust, amely kiírja az összes könyvet a könyvtárban!
