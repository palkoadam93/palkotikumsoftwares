Fájlkezelés

SuliPY - JSON és Python

Feladatok

1. Írj egy Python programot, amely létrehoz egy új JSON fájlt, és beleír egy szótárt, amely tartalmaz néhány kulcs-érték párt. A program ezután olvassa be a fájl tartalmát, és írja ki a képernyőre!

2. Készíts egy Python programot, amely egy meglévő JSON fájlba hozzáfűz egy új kulcs-érték párt. A program ellenőrizze, hogy a fájl létezik-e, és ha nem, hozzon létre egy új fájlt!

3. Írj egy Python programot, amely beolvassa egy JSON fájl tartalmát, és számolja meg, hány kulcs található a szótárban. A program írja ki ezt az információt!

4. Készíts egy Python programot, amely egy JSON fájlt olvas be, és az értékek alapján csoportosítja a kulcsokat. Például, ha az értékek listák, akkor azokat egy listában gyűjti össze!

5. Írj egy Python programot, amely egy JSON fájlba ír be adatokat. A program kérjen be adatokat a felhasználótól, és minden új adatot új kulcs-érték párként írjon a fájlba!

6. Készíts egy Python programot, amely beolvassa egy JSON fájl tartalmát, és minden szótárbejegyzést egy új JSON fájlba ír, ahol a kulcsok az eredeti fájl kulcsai!

7. Írj egy Python programot, amely egy JSON fájl tartalmát beolvassa, és minden bejegyzést egy listába helyez el. A program írja ki a listát!

8. Készíts egy Python programot, amely egy JSON fájl adatait módosítja. A program kérje be a felhasználótól, hogy melyik kulcs értékét kell módosítani, és írja át a megadott értéket a fájlban!

9. Írj egy Python programot, amely egy JSON fájl minden kulcsában keres egy adott szót, és írja ki azokat a kulcsokat, amelyek tartalmazzák a keresett szót!

10. Készíts egy Python programot, amely egy JSON fájl adatait rendezi az egyik kulcs értékei alapján. A program kérje be a kulcs nevét, és az alapján rendezze a bejegyzéseket a fájlban!
