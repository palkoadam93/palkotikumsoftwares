Feladatok

1. Írj egy Python programot, amely próbál megnyitni egy nem létező fájlt. Használj try és except blokkokat a kivétel kezelésére, és írd ki egy hibaüzenetet, ha a fájl nem található!

2. Készíts egy Python programot, amely próbál osztani két számot. Ha a második szám nulla, kezelj egy ZeroDivisionError kivételt, és írj ki egy hibaüzenetet a képernyőre!

3. Írj egy Python programot, amely egy felhasználótól kér be egy számot, és próbálja azt egész számra konvertálni. Használj try és except blokkokat a ValueError kivétel kezelésére, ha a felhasználó nem érvényes számot ad meg!

4. Készíts egy Python programot, amely egy JSON fájl olvasásakor próbálkozik a fájlformátum ellenőrzésével. Használj try és except blokkokat a json.JSONDecodeError kivétel kezelésére, és írd ki, ha a fájl nem érvényes JSON!

5. Írj egy Python programot, amely egy listából próbál hozzáférni egy indexhez. Ha az index érvénytelen, használj try és except blokkokat az IndexError kivétel kezelésére, és írd ki egy hibaüzenetet!

6. Készíts egy Python programot, amely próbál hozzáadni egy elemhez egy nem létező listát. Használj try és except blokkokat a AttributeError kivétel kezelésére, és írd ki a megfelelő hibaüzenetet!

7. Írj egy Python programot, amely egy függvény hívásakor kezel egy általad generált kivételt. Használj raise a kivétel generálására, és try és except blokkokat a kezelésére!

8. Készíts egy Python programot, amely próbál megnyitni egy fájlt írásra, de a fájl írásához nem rendelkezik jogosultsággal. Használj try és except blokkokat az IOError kivétel kezelésére, és írd ki a megfelelő hibaüzenetet!

9. Írj egy Python programot, amely egy szótár értékének hozzáférésekor próbál hozzáférni egy nem létező kulcshoz. Használj try és except blokkokat a KeyError kivétel kezelésére, és írd ki a megfelelő hibaüzenetet!

10. Készíts egy Python programot, amely egy matematikai művelet során próbál egy nem létező változót használni. Használj try és except blokkokat a NameError kivétel kezelésére, és írd ki a megfelelő hibaüzenetet!
