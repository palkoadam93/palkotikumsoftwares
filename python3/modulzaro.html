Python modulzáró feladat

Feladat:

Készíts egy Python programot, amely az alábbi funkciókat valósítja meg:

1. Fájlkezelés:

- Definiálj egy Library osztályt, amely kezeli a könyveket. A könyvek adatait egy JSON fájlban tárold, amely tartalmazza a könyv címét, szerzőjét és ISBN számát. Az osztály rendelkezzen az alábbi metódusokkal:

- add_book(title, author, isbn): hozzáad egy új könyvet a JSON fájlhoz.

- remove_book(isbn): eltávolít egy könyvet az ISBN szám alapján.

- list_books(): kiírja az összes könyvet a JSON fájlból.

2. Kivételkezelés:

- Gondoskodj arról, hogy a fájlműveletek során megfelelő kivételkezelést végezz, például:

- Ha a JSON fájl nem található, hozz létre egy új fájlt.

- Ha a JSON fájl érvénytelen, kezelj egy json.JSONDecodeError kivételt.

- Ha egy könyv törlése során az ISBN szám nem létezik, kezelj egy KeyError kivételt.

3. Modulok és csomagok:

- Használj modulokat és csomagokat a program különböző részeinek strukturálására. Például, a fájlkezelést és a kivételkezelést külön modulokban valósíthatod meg, és importálhatod őket az alapvető könyvtárkezelő osztályba.

4. Objektum-orientált programozás:

- Az egész programot objektum-orientált módon alakítsd ki. Az osztályok és metódusok használatával oldd meg a feladatokat, és ügyelj a megfelelő öröklődés és polimorfizmus alkalmazására, ha szükséges.

Példa JSON fájl tartalomra:

    [
        {
            "title": "Python Programming",
            "author": "John Doe",
            "isbn": "1234567890"
        },
        {
            "title": "Advanced Python",
            "author": "Jane Smith",
            "isbn": "0987654321"
        }
    ]

Megjegyzés: Ügyelj a kód tisztaságára és dokumentációjára, valamint a megfelelő tesztelésre és hibakezelésre!
